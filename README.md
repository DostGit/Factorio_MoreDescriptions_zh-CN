Adds flavored and basic descriptions to almost every item and entity in the game, including Space Age content! This is a very niche mod, but it makes the game feel more alive in my opinion. Yes, now every fluid barrel tells you that it is, in fact, a barrel filled with a fluid. See the gallery for other examples.

### Features
* Descriptions for hundreds of items and entities.
* A mix of useful tidbits, "lore"-friendly flavor, and a pinch of dry humor.
* Full support for Space Age.
